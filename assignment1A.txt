IMDB 5000 movies dataset is a dataset that record over 5 thousands of the data which about top movies, documentation, and drama that is listed on IMDB.com. The data type in this dataset is in table form, and this dataset contain document and network data. In this datasets there are 28 columns, where each column represent the attribute for the objects (rows). Both categorical (eg. color of movie) and numeric attribute exist in this dataset for each column/ attribute. Each row represent a single movie/documentation/drama, and there are 5043 rows in this dataset which means this raw dataset contain about data for 5043 movies.
